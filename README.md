# OpenCode-Agentic-Orchestration

## Overview

OpenCode-Agentic-Orchestration is a structured, three-tier agentic framework that builds on ECC for disciplined, high-quality automation.

## Architecture

The project enforces a strict three-tier separation:
- Intelligence (`.opencode/`): Global AI behavior, slash commands, agent definitions
- Context (`plan/`): Project roadmap, conventions, instructions
- Implementation (`codebase/`): The ONLY location for all application source code and Git repository

Phase 0 setup complete, including standard repository scaffold, schemas, skills library, opencode.json root-relative path mapping, and core automation scripts.
